# M-t-oB-nin

## Description
Application Python qui récupère des données météo réelles
via l'API Open-Meteo, les stocke en JSON et affiche des
statistiques comparatives entre plusieurs villes.

## Équipe
- AlfredBOCOVO 
- stephane964 

## Fonctionnalités
- Recherche météo par ville (ex : Cotonou)
- Affichage température min / max / moyenne
- Comparaison entre deux villes
- Sauvegarde des données en JSON

## Technologies
Python | API Open-Meteo | Git | Pytest | PEP 8

## Installation
git clone https://github.com/AlfredBOCOVO/M-t-oB-nin.git
cd M-t-oB-nin
python -m venv venv
pip install -r requirements.txt
